登录门禁 Auth 真实

团队成员即账号，挂在 workspace（租户）下，登录后自动切租户。

强制登录门禁

控制台强制登录：团队成员（IAM users）即登录账号，挂在 workspace（租户）下；邮箱 + 密码登录，登录后 workspace 切到该用户租户，右上角显示用户 + 登出。
