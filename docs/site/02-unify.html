功能英文状态说明

OneID 归一原理

ID-Mapping 服务（:8001）消费多渠道 UserEvent（微信 openid/unionid、企微 extid、手机号、邮箱、设备），解析 / 合并身份为一个 OneID：热映射写 Redis（<5ms 点查），审计与画像写 MySQL（id_mapping / merge_log / user_profile）。
