核心能力

对话即可操作的数据平台

控制台为对话形态：左侧会话 · 中间智能助手（Ask / Agent）· 右侧实时数据底座与知识库。

统一 Unify · OneID

多渠道实时归一为统一画像

小程序 / 企微 / 表单 / App 等多渠道事件实时合并为一个 OneID，打宽成画像宽表，向上提供可检索的用户档案与聚合指标。

  • 热映射 Redis <5ms，审计与画像落库
  • 右栏「数据底座」实时概览，点数字下钻
触达 Engage · 自然语言圈人

一句话圈人，安全可控

中文描述即可圈选人群，实时预估人数并预览生成的查询。LLM 绝不直出 SQL——候选 DSL 必须过校验层。

  • 多条件 / AND·OR / 跨对象多跳关联
  • 预估 → 保存为受众 → 激活到下游
知识库 · 卡帕西 LLM 模式

分域知识 + LLM 上下文策展

上下文 = RAM、文件 = 磁盘：分域文件夹承载多模态资料，文档为主干，一键把文件/整域纳入 LLM 上下文，实时估算 token 预算。

  • 多模态：文档 / 图片 / 视频 / 音频 / 文件
  • 可关联到对象记录，支撑归因分析
分析 Analyst · NL 生成看板

说一句话，生成看板

描述一句话即从白名单数据源受限选图生成看板/图表（LLM 不写 SQL），图表数据点可下钻明细。

  • 内置画像 / 转化率 ROI 看板
  • recharts 安全聚合，口径受控
