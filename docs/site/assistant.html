智能助手 Assistant 真实

多智能体：数据查询(MCP) / 分析(建看板) / 任务 / 通用导航，外加主动式埋点 Copilot。

41 MCP 工具多智能体主动 Copilot

多智能体架构

控制台右上角常驻对话助手：路由器先判定意图，分派给专职智能体，各有独立系统提示与工具集。服务独立（:8004），DeepSeek 不可用时降级。

🔎数据查询

桥接 41 个只读 MCP 工具（查 schema / 受众 / 对象 / 画像…）。

📊分析

自然语言创建图表 / 看板（调 /analyst/*/nl 落库，回复带「打开看板」直链）。

🗂️任务

「发布任务」落到 reverse-ETL 调度模拟后台运行。

🧭通用 + 导航

产品答疑 + 页面导航（默认加载 page-routes.md，说「打开 X 页面」即 open_page 跳转）。

主动式埋点 Copilot

助手不只被动等问：前端埋点采集左侧页面行为，近实时推给助手（POST /observe），经启发式门控 + LLM 判断后主动弹出建议（任意页面长时间停留主动问询、空结果建议放宽条件、报错帮排查）。两层门控控成本，只产出「建议 + action」不自动写，侧边栏铃铛可免打扰。
