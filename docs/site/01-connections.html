功能英文状态说明
数据源Sources真实数据源卡片列表；可视化 ETL：CSV/粘贴 → 字段映射(自动/手动) → 预览 → 导入到任意对象，可建关系
数据源目录Source Catalog真实44 个主流连接器平铺成卡片（8 类：数据库/数仓/数据湖/查询引擎/对象存储/流/文件/API），点卡片填名即建数据源（MySQL/PostgreSQL/ClickHouse/MongoDB/Snowflake/BigQuery/Iceberg/Delta/S3/Kafka/Trino…，连接逻辑为占位适配器）
可视化编排Pipelines (Flow)真实拖拽节点连线编排 ETL → 保存为管道
管道Pipelines真实管道列表 → 详情；点「执行」经 Apache Airflow 真实触发 DAG（动态多任务展开）；详情含拓扑、执行历史、暂停/恢复调度
数据源详情Source DetailMockWrite Key、Schema 事件、实时事件流 Debugger
目的地DestinationsMock广告/MA/分析/Webhook 目录，把数据激活到下游
Reverse ETLReverse ETL真实以数仓为源的任务，run-now 触发 + 运行记录（调度模拟）
数据仓库Warehouses真实连接器目录连数仓（Doris/Snowflake/Iceberg…），同步状态
FunctionsFunctionsMock自定义代码在数据源/目的地侧转换数据

可视化编排 → 真实 Airflow 调度

「可视化编排」拖拽节点连线保存为管道，在「管道」页点执行 → SQL Engine 经 Airflow REST API 触发参数化通用 DAG（plan → run_node×N → finish，动态任务映射 .expand）。Airflow 不可达时优雅降级为本地模拟。详见 安装与使用。
