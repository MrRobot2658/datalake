圈人规则

统一筛选器支持 ① 多条件 + AND/OR ② 跨对象链式多跳关联（用户 → 下单 → 商品）③ 关系边条件 ④ 自然语言（中文描述 → DSL）。查询前可「预估人数」并预览生成的 SQL，满意后「存为受众」。LLM 绝不直出 SQL，候选 DSL 必须过 dsl.py 校验层（字段/操作符/关系/跳数）。
