DataAgent · 企业级智能体系统架构

CDP + Multi-Agent Platform — 实时 & 离线双引擎 · 语义驱动 · Agent 自治

实时处理链路

  • • 数据源 CDC → Debezium → Kafka → Flink 流处理
  • • Flink 实时写入 Doris 宽表 + 触发 CEP 规则
  • • 语义事件 → Agent 实时响应（<1s）
  • • 端到端延迟：数据变更到 Agent 感知 <5s

离线处理链路

  • • Airflow DAG 调度 T+1 批处理任务
  • • Spark 大批量 ETL + 数据清洗 + 特征工程
  • • dbt 模型转换 + 数据质量校验 + 血缘追踪
  • • 结果写入 Doris/PostgreSQL → 语义层消费

语义驱动架构

  • • 所有数据源 → 统一 Ontology 建模
  • • UQL 替代原生 SQL，跨系统查询一条搞定
  • • Agent 只感知语义层，不感知底层存储
  • • 系统边界消融：CRM/ERP 退化为连接器
